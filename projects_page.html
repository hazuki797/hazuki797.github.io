Projects


Hello and welcome to my coding project showcase! I am excited to present to you the web development skills that I have acquired through the MIT xPRO full stack programming coding bootcamp.

Each project has been carefully crafted to showcase my abilities in web development, such as my proficiency in JavaScript, HTML and CSS, and my ability to create interactive web applications. This projects section showcases some of the web applications that I have created during this course. Each project is accompanied by a brief description and a link to the project and the Github repository where you can find the code that I have written.

Overall, I am proud of the work that I have done on this project, and I hope that you find it informative and engaging. Thank you for taking the time to view my personal portfolio, and I look forward to hearing your feedback!

Eye Movement

The Eye Movement project allows you to move the dog's eye movement as it follows your cursor. This simple yet charming project is my favorite exercise.



Pac-Man

The Pac-Man project will show one big pac-man moving across the screen and addition of mini pac-mans with a click of a button. The large Pac-man changes direction when it reaches the boundaries of the webpage. The mini Pac-mans moves randomly upon starting the game, with a random image and location chosen upon creation.


Bus Tracker

The bus tracker project shows the bus stops between MIT and Harvard.This project showcases the use of Mapbox API in integration of HTML, CSS, and JavaScript to create a dynamic experience for the user.
